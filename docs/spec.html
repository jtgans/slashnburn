General Requirements

All code written for SNB must conform to the following requirements:

  • Simple
  • Easy to read
  • Elegant (eg: solves the coding problem in a clean, understandable fashion)
  • Commented via PHPDOC

All XHTML code output to the browser must conform to the following requirements:

  • XHTML 1.0 Strict
  • No attempt to change layout or design is made (leave this to the theme designer)
  • Focus is on content, not markup
  • Markup is clean and concise

Architecture Overview

The SNB system is designed in an almost zen fashion -- each part of it performs a distinct function in the most elegant way possible. The core of the system is composed of several smaller parts:

Navigation
The navigation system is simply the nav bar which is everpresent on every page that SNB displays. The navigation bar is completely customizable, but attempts to keep track of which page the user is on.
Templating / Theming
The templating system is composed entirely of the Smarty template engine. Each site theme is composed of a simple header, footer, CSS and images. All other code (other than the header and footer) is currently not customizable, except via CSS.
