What areas of design themes use

Themes consist of a single CSS file and many optional images. The CSS must validate at level 1 conformance. None of the XHTML markup used to generate the pages may be changed. Yes, this is strict, yes it's a deliberate design decision. Attempting to create pages for each module in a theme is too complex and too wide a scope for SNB to solve. Therefore, if you want to change the way your site looks, just change the CSS code. Virtually all parts of the system have different <div> ids, and can be rearranged to your heart's content.

Basic-level conformance

Throughout the entire system, there are three basic modules that must have CSS code written for them. These modules are the Blog, Articles, and Links. Any other modules are optional for the theme to support.
